Transor Immersive Translation

A powerful web translation tool that allows you to browse global websites in your preferred language

Key Features

Multiple Translation Styles

Supports inline, replace, bilingual, hover and more translation styles to meet different reading scenarios

Multiple Translation Engines

Supports Google, DeepL, Microsoft and other translation engines to get the most accurate results

Smart Filtering

Automatically skips code, scripts, and other content that doesn't need translation, maintaining the original functionality

Highly Customizable

Customize CSS, exclude specific tags and class names, and have complete control over the translation effect

How to Use

  1. Click the Transor icon in the Chrome toolbar to open the settings panel
  2. Select your target language, source language, translation engine, and display style
  3. Turn on the translation toggle, and the current page will be automatically translated
  4. Adjust advanced settings as needed, such as excluding specific tags or classes
  5. Enjoy a seamless multilingual reading experience!
